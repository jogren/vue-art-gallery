# Vue Art Gallery

Vue Art Gallery is the result of a code challenge to learn Vue.js in just 2 days and create an art gallery using the Harvard Museums API.

## Tech Stack
- Vue.js
- JavaScript(ES6+)
- CSS3
- Testing with Jest

## Project Setup
```
git clone https://github.com/jogren/vue-art-gallery
npm install
npm run serve
```

## Run Tests
```
npm test
```
